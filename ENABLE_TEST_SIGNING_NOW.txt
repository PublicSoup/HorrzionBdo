===============================================
  ENABLE TEST SIGNING - DO THIS NOW
===============================================

YOUR DRIVER IS READY!
Just needs test signing enabled.

===============================================
STEP 1: Open PowerShell as Administrator
===============================================

1. Press Windows key
2. Type: PowerShell
3. Right-click "Windows PowerShell"
4. Click "Run as Administrator"

===============================================
STEP 2: Run This Command
===============================================

Copy and paste this into the PowerShell window:

bcdedit /set testsigning on

Press Enter

===============================================
STEP 3: Restart Computer
===============================================

Type:
shutdown /r /t 0

Or just restart normally from Start menu

===============================================
STEP 4: After Restart
===============================================

Come back to this folder and run:

START_EVERYTHING.bat

(Right-click → Run as Administrator)

===============================================
THAT'S IT!
===============================================

Your driver will load and scanner will work!
