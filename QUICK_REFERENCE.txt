================================================================================
1. Build Driver (Visual Studio + WDK):
   Open BDO_KernelDriver.c in Driver project → Build

2. Enable Test Signing:
   bcdedit /set testsigning on
   shutdown /r /t 0

3. Load Driver:
   sc create BDOKernelMem type=kernel binPath="C:\Full\Path\BDO_KernelDriver.sys"
   sc start BDOKernelMem

4. Build & Run Client:
   cl /EHsc /std:c++17 BDO_KernelClient.cpp BDO_Example.cpp

🐛 TROUBLESHOOTING
================================================================================
Driver won't load:
  → Enable test signing: bcdedit /set testsigning on

Access denied:
  → Run as Administrator
  → Check driver is started: sc query BDOKernelMem

Reads fail:
  → Call BypassAntiCheat() first
  → Verify offsets are current
  → Check process ID is correct

BSOD:
  → Safe Mode: F8 on boot
  → Delete driver: sc delete BDOKernelMem
  → Check DbgView logs
