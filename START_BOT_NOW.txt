============================================================
📖 STEP-BY-STEP FOR ABSOLUTE BEGINNERS
============================================================

STEP 1: First Time Setup
  1. Find file: RUN_EVERYTHING.bat
  2. Right-click it
  3. Select: "Run as administrator"
  4. Click: "Yes" on the popup
  5. Follow the prompts
  6. Wait for compilation (1-2 minutes)
  7. Bot will launch automatically!

STEP 2: Using the Bot (First Time)
  1. Start Black Desert Online
  2. Log into your character
  3. In the bot window, choose: 1
     (This scans for addresses automatically)
  4. Wait 30 seconds
  5. Addresses found!
  6. Choose: 4 (Combat Bot)
  7. Bot is now running!

STEP 3: Daily Use (After First Time)
  1. Start Black Desert Online
  2. Log into character
  3. Double-click: ONE_CLICK_START.bat
  4. In bot window, choose: 3
     (This loads your saved addresses)
  5. Choose: 4 (Combat Bot)
  6. Done!

🆘 HELP! SOMETHING WENT WRONG
============================================================

PROBLEM: "Visual Studio not found"
  → Install Visual Studio 2019 or 2022
  → Download: https://visualstudio.microsoft.com/

PROBLEM: "Bot closes immediately"
  → Start BDO first!
  → Read: WHY_BOT_CLOSES.md

PROBLEM: "Compilation failed"
  → Try: COMPILE_SIMPLE.bat
  → Or open BDOStealthBot.sln in Visual Studio

PROBLEM: "Access Denied"
  → Right-click → Run as administrator
