HOW IT WORKS:
================================================================================

Simple version:
  1. Finds Black Desert process
  2. Opens handle to read memory
  3. Uses ReadProcessMemory()
  4. Displays the data

Technical version:
  1. CreateToolhelp32Snapshot() to find process
  2. OpenProcess(PROCESS_VM_READ) for handle
  3. Pattern scan for dynamic addresses
  4. Follow pointer chains
  5. Read game structures
  6. Parse and display data

See HOW_IT_WORKS.txt for visual diagrams!
