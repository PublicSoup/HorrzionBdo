OPTION 3: USE CHEAT ENGINE (MOST RELIABLE)
================================================================================

This is the "real" way to find offsets:

1. Download Cheat Engine (free)
2. File → Open Process → BlackDesert64.exe
3. Scan Type: Float
4. Enter your current HP value (e.g., 2543)
5. First Scan
6. Take damage in game
7. Enter new HP value
8. Next Scan
9. Repeat until you have 1-5 addresses
10. Right-click one → "Find out what accesses this address"
11. Play game for a bit to trigger the code
12. You'll see assembly instructions
13. Copy the pattern from the instruction bytes

Example result:
  BlackDesert64.exe+A1B2C8 - 48 8B 0D 12 34 56 78 - mov rcx,[rip+78563412]
  
Pattern to use:
  48 8B 0D ?? ?? ?? ?? 

Then update BDOMemory.cpp with the new pattern!

WHY PATTERNS CHANGE:
================================================================================

Every BDO patch:
- Game code gets recompiled
- Addresses shift around
- Static pointers move
- Offsets stay the same, but base addresses change

That's why we use pattern scanning instead of hard-coded addresses!

Pattern: 48 8B 0D ?? ?? ?? ??
This matches: MOV RCX, [RIP+offset]

The instruction pattern stays the same, but the actual address changes.
